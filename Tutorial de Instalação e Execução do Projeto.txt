## Guia Completo: Instalação e Execução do Projeto

Este tutorial foi feito para te ajudar a configurar e rodar o projeto do início ao fim. Siga os passos e, em pouco tempo, a aplicação estará funcionando no seu computador.

### Pré-requisitos

Antes de começar, certifique-se de que você tem as seguintes ferramentas instaladas:

  * **[Python 3.x](https://www.python.org/downloads/)**
  * **[Git](https://git-scm.com/downloads)**
  * **[MongoDB](https://www.mongodb.com/try/download/community)** (certifique-se de que o servidor está rodando)

-----

### Passo 1: Clonar o Repositório

Abra seu terminal (como o PowerShell no Windows) e use o Git para copiar o projeto para o seu computador.


git clone https://github.com/yuriiSw/PROJETO-AlertaMed.git


-----

### Passo 2: Acessar a Pasta do Projeto

Entre na pasta do projeto que você acabou de clonar:


cd nome-do-repositorio


-----

### Passo 3: Configurar o Ambiente Virtual

Criar um ambiente virtual isola as dependências do projeto, evitando conflitos com outras aplicações Python.

1.  **Crie o ambiente virtual:**

    
    python -m venv venv
    

2.  **Ative o ambiente virtual:**

      * **No Windows (PowerShell):**

        
        .\venv\Scripts\activate
        

        Se aparecer um erro de segurança, siga estes passos:

          * Abra o **PowerShell como Administrador**.
          * Execute o comando: `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser`
          * Digite `S` para confirmar.
          * Feche e abra um novo terminal para tentar ativar o ambiente virtual novamente.

      * **No macOS e Linux:**

        
        source venv/bin/activate
        

Você saberá que o ambiente está ativo quando o texto `(venv)` aparecer no início da linha de comando.

-----

### Passo 4: Instalar as Dependências

Com o ambiente virtual ativo, instale todas as bibliotecas necessárias para o projeto:


pip install -r requirements.txt


-----

### Passo 5: Executar a Aplicação

Agora é só iniciar o servidor Flask. O arquivo principal da aplicação é o `app.py`.


python app.py


O terminal vai mostrar uma mensagem indicando que a aplicação está rodando, geralmente no endereço `http://127.0.0.1:5000`.

-----

### Passo 6: Acessar a Aplicação

Abra seu navegador e acesse:

`http://127.0.0.1:5000`

A aplicação deve estar funcionando perfeitamente.

-----

### Dicas Extras

  * Para **desativar** o ambiente virtual, use o comando: `deactivate`.
  * Se precisar de ajuda com o MongoDB, verifique a string de conexão no arquivo principal do projeto (`app.py`). Ela deve estar parecida com `"mongodb://localhost:27017/nome-do-banco-de-dados"`.
